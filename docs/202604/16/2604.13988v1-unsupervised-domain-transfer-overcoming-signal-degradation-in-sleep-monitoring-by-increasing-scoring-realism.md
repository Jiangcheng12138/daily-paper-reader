## 摘要
目的：探讨睡眠图的“真实性”是否可用于指导一种无监督方法，以处理移动睡眠监测中任意类型的信号退化。方法：将预训练的最先进“u-sleep”模型与“判别器”网络相结合，我们将目标域的特征与预训练期间学习到的特征空间进行对齐。为了测试该方法，我们使用真实的信号退化对源域进行失真处理，以观察该方法对不同退化类型的适应程度。我们将所得模型的性能与针对每种迁移类型以有监督方式设计的最佳情况模型进行了比较。主要结果：根据失真类型的不同，我们发现该无监督方法可以将 Cohen's kappa 系数提高 0.03 到 0.29 不等，且在所有迁移任务中，该方法均未降低性能。然而，该方法从未完全达到估计的理论最优性能，且在针对两项睡眠研究之间的现实领域失配进行测试时，其收益并不显著。意义：“判别器引导的微调”是处理“自然场景下”睡眠监测信号退化的一种有趣且具有一定前景的方法。特别是，它对一般睡眠数据的揭示具有研究价值。然而，在将其投入“实际应用”之前，仍需进一步开发。

## Abstract
Objective: Investigate whether hypnogram 'realism' can be used to guide an unsupervised method for handling arbitrary types of signal degradation in mobile sleep monitoring.   Approach: Combining a pretrained, state-of-the-art 'u-sleep' model with a 'discriminator' network, we align features from a target domain with a feature space learned during pretraining. To test the approach, we distort the source domain with realistic signal degradations, to see how well the method can adapt to different types of degradation. We compare the performance of the resulting model with best-case models designed in a supervised manner for each type of transfer.   Main Results: Depending on the type of distortion, we find that the unsupervised approach can increase Cohen's kappa with as little as 0.03 and up to 0.29, and that for all transfers, the method does not decrease performance. However, the approach never quite reaches the estimated theoretical optimal performance, and when tested on a real-life domain mismatch between two sleep studies, the benefit was insignificant.   Significance: 'Discriminator-guided fine tuning' is an interesting approach to handling signal degradation for 'in the wild' sleep monitoring, with some promise. In particular, what it says about sleep data in general is interesting. However, more development will be necessary before using it 'in production'.

*   **研究背景**：自动睡眠评分模型在实验室环境（PSG金标准）下表现优异，但在“野外”环境（如移动监测、干电极EEG）中，由于信号退化（噪声、干扰、设备差异）导致性能大幅下降。
*   **核心问题**：如何在**没有目标领域人工标注标签**的情况下，修复模型在受损信号上的评分准确性？
*   **核心思想**：作者提出利用睡眠图（Hypnogram）的“真实性”（Realism）。正确的睡眠分期序列具有特定的时序规律（如阶段持续时间、转换顺序），而受损信号生成的评分往往杂乱无章。通过对抗学习迫使模型生成“看起来更真实”的评分，从而间接提升准确率。

### 2. 方法论
*   **核心架构**：采用“判别器引导的微调”（Discriminator-guided fine-tuning）框架。
    *   **睡眠评分网络 (U-Sleep)**：分为编码器（Encoder）和解码器（Decoder）。
    *   **判别器 (Discriminator)**：一个轻量级的 Transformer 结构，输入是模型预测的 5 类睡眠阶段的概率分布序列。
*   **算法流程**：
    1.  **初始化**：使用在高质量数据集上预训练好的 U-Sleep 模型。
    2.  **对抗训练**：
        *   **判别器**：学习区分“来自干净源域的评分分布”和“来自退化目标域的评分分布”。
        *   **目标域编码器 ($E_t$)**：通过微调，尝试欺骗判别器，使其认为退化信号生成的评分分布与干净信号一致。
    3.  **损失函数**：
        *   **对抗损失 ($L_{adv}$)**：驱动目标域特征分布向源域对齐。
        *   **锚点损失 ($L_{anchor}$)**：惩罚微调后的编码器与原始预训练编码器在相同输入下的偏差，防止模型参数过度漂移。
*   **技术特点**：该方法不尝试“修复”原始信号（去噪），而是直接优化下游任务（评分）的输出逻辑。

*   **训练细节**：Batch size 设为 1（即单次输入一整晚的记录），学习率为 $10^{-5}$（编码器）和 $10^{-6}$（判别器），训练 50 个 epoch。

### 5. 实验数量与充分性
*   **实验规模**：使用了近 4000 晚的数据，样本量充足。
*   **实验维度**：
    *   针对三种失真类型分别进行了测试。
    *   进行了**数据规模缩放实验**（从 10% 到 90% 的训练量），探讨数据量对性能的影响。
    *   进行了**消融实验**，验证了锚点损失、判别器数量、位置编码等设计的重要性。
    *   引入了 **Chance kappa 分析**，证明性能提升并非源于模型生成了某种“通用模板”，而是确实基于输入信号。

### 6. 主要结论与发现
*   **性能提升显著**：在所有模拟失真场景下，该方法均提升了评分表现，Kappa 值提升幅度在 **0.03 到 0.29** 之间。
*   **未达上限**：虽然显著优于原始模型，但性能始终未能完全达到有监督训练的基准水平。
*   **真实场景挑战**：在尝试从源域迁移到完全不同的真实数据集（Tabar et al.）时，提升并不显著。作者认为这可能与目标域数据量过小（仅 120 条记录）有关。
*   **数据依赖性**：该方法对数据量有较高要求，性能随训练样本增加而持续增长，但在数千条记录后出现边际效应。

### 7. 优点
*   **无需标签**：解决了移动监测中人工标注成本极高的问题。
*   **任务导向**：避开了复杂的信号去噪过程，直接利用睡眠分期的生理规律（真实性）作为约束。
*   **通用性强**：实验证明该框架可以处理多种不同性质的信号退化。

### 8. 不足与局限
*   **数据饥渴**：目前需要数千条记录才能实现有效微调，这在某些小众临床研究中难以实现。
*   **真实域迁移乏力**：在模拟噪声上效果极佳，但在处理复杂的真实设备间差异时，表现尚需证明。
*   **架构依赖**：该方法高度依赖于预训练模型的质量，且目前仅在 U-Sleep 架构上进行了验证。
