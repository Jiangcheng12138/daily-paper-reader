## 摘要
目的：探讨睡眠图的“真实性”是否可用于指导一种无监督方法，以处理移动睡眠监测中任意类型的信号退化。方法：将预训练的最先进“u-sleep”模型与“判别器”网络相结合，我们将目标域的特征与预训练期间学习到的特征空间进行对齐。为了测试该方法，我们使用真实的信号退化对源域进行失真处理，以观察该方法对不同类型退化的适应能力。我们将所得模型的性能与针对每种迁移任务以有监督方式设计的最佳情况模型进行了比较。主要结果：根据失真类型的不同，我们发现无监督方法可以将 Cohen's kappa 系数提高 0.03 到 0.29 不等，且对于所有迁移任务，该方法均未降低性能。然而，该方法从未完全达到估计的理论最优性能，且在针对两项睡眠研究之间的现实领域失配进行测试时，其收益并不显著。意义：“判别器引导的微调”是处理“自然场景下”睡眠监测中信号退化的一种有趣且具有一定前景的方法。特别是，它对一般睡眠数据的揭示具有重要意义。然而，在将其投入“实际应用”之前，仍需进一步开发。

## Abstract
Objective: Investigate whether hypnogram 'realism' can be used to guide an unsupervised method for handling arbitrary types of signal degradation in mobile sleep monitoring.   Approach: Combining a pretrained, state-of-the-art 'u-sleep' model with a 'discriminator' network, we align features from a target domain with a feature space learned during pretraining. To test the approach, we distort the source domain with realistic signal degradations, to see how well the method can adapt to different types of degradation. We compare the performance of the resulting model with best-case models designed in a supervised manner for each type of transfer.   Main Results: Depending on the type of distortion, we find that the unsupervised approach can increase Cohen's kappa with as little as 0.03 and up to 0.29, and that for all transfers, the method does not decrease performance. However, the approach never quite reaches the estimated theoretical optimal performance, and when tested on a real-life domain mismatch between two sleep studies, the benefit was insignificant.   Significance: 'Discriminator-guided fine tuning' is an interesting approach to handling signal degradation for 'in the wild' sleep monitoring, with some promise. In particular, what it says about sleep data in general is interesting. However, more development will be necessary before using it 'in production'.

### 1. 核心问题与整体含义
*   **研究动机**：在“野外”或移动场景（如干电极耳部EEG）进行睡眠监测时，信号质量往往因电极接触不良、环境噪声或硬件限制而发生严重退化。
*   **核心问题**：现有的自动睡眠评分模型在面对这种“领域失配”（Domain Mismatch）时表现不佳。作者提出，人类专家在评分时会利用睡眠图（Hypnogram）的“真实性”（Realism，即睡眠阶段的顺序和持续时间规律）来过滤噪声。本研究旨在探索能否通过**无监督学习**，利用这种“真实性”作为引导，使模型自动适应各种信号退化。

### 2. 方法论
*   **核心思想**：采用**对抗性领域迁移（Adversarial Domain Transfer）**框架。通过一个判别器来区分“真实的睡眠评分模式”（来自干净的源域）和“受损的评分模式”（来自退化的目标域），迫使编码器学习如何从噪声信号中提取能生成“真实睡眠图”的特征。
*   **关键技术细节**：
    *   **基础模型**：使用预训练的 **U-Sleep** 模型作为骨干，将其拆分为编码器（Encoder）和解码器（Decoder）。
    *   **判别器引导微调（Discriminator-guided fine tuning）**：
        *   **源域编码器 ($E_s$)**：冻结，提供干净信号的特征基准。
        *   **目标域编码器 ($E_t$)**：可学习，负责处理退化信号。
        *   **解码器 ($Dec$)**：冻结，确保输出空间的一致性。
        *   **判别器 ($D$)**：基于 Transformer 的轻量级分类器，输入是预测的睡眠阶段概率分布，而非原始信号。
    *   **损失函数**：
        *   **对抗损失 ($L_{adv}$)**：通过 Min-Max 博弈，使 $E_t$ 生成的特征在判别器眼中与 $E_s$ 生成的特征不可区分。
        *   **锚点损失 ($L_{anchor}$)**：惩罚 $E_t$ 与 $E_s$ 在相同输入下的特征偏差，确保模型在处理非退化数据时仍保持准确（协变量偏移假设）。

### 3. 实验设计
*   **数据集**：使用了来自 National Sleep Research Resource 的 18 个公开数据集（如 ABC, CCSHS, MASC 等），涵盖数千名受试者。
*   **失真场景（目标域）**：
    1.  **白噪声**：模拟通道损坏。
    2.  **放大器过载**：模拟电极连接不良产生的双指数脉冲尖峰。
    3.  **频谱变形**：通过不同频率的带通滤波器模拟硬件或生理导致的频率分布改变。
*   **Benchmark（基准）**：
    *   **Pretrained**：未经微调的原始 U-Sleep 模型。
    *   **Supervised Benchmark**：在有标签的目标域数据上进行有监督训练的“理论上限”模型。
    *   **Chance Kappa**：通过随机配对睡眠图计算的偶然一致性水平。

### 4. 资源与算力
*   **硬件**：实验在一台带有**专用 GPU 的笔记本电脑**上完成。
*   **训练时长**：单次运行（50 个 Epoch）耗时约 **6 到 8 小时**。
*   **批大小**：Batch Size 为 1（即整晚记录），以适应笔记本显存并保持训练稳定。

### 5. 实验数量与充分性
*   **实验规模**：使用了 **3,931 条睡眠记录**，其中 80% 用于训练，20% 用于测试。
*   **实验维度**：涵盖了三种主要的信号失真类型，并对失真程度（如滤波频率、脉冲频率）进行了参数化扫描。
*   **消融实验**：进行了消融研究，验证了锚点损失、判别器数量、Transformer 架构组件（如位置编码）的必要性。
*   **充分性评价**：实验设计较为客观，通过“Chance Kappa”分析排除了模型仅生成“通用模板”的可能性。但在真实跨研究领域（非人工模拟失真）的实验较少，且效果不显著。

### 6. 主要结论与发现
*   **性能提升**：在所有模拟失真场景下，该方法均显著优于预训练模型。Cohen’s Kappa 提升幅度在 **0.03 到 0.29** 之间。
*   **真实性引导有效**：证明了睡眠图的“稀疏性”（即只有极少数序列是符合生理规律的）可以作为无监督学习的有效约束。
*   **局限性**：虽然提升显著，但仍未达到有监督训练的性能水平。在样本量较小的真实领域迁移（如 Tabar 数据集）中，由于数据量不足（仅 120 条记录），提升不明显。

### 7. 优点
*   **无需标签**：在目标域完全不需要人工标注，极具临床应用潜力。
*   **通用性**：不针对特定噪声建模，而是通过“结果真实性”反向约束特征提取，能应对多种未知的信号退化。
*   **稳定性**：引入锚点损失确保了模型在干净数据上的鲁棒性，避免了灾难性遗忘。

### 8. 不足与局限
*   **数据依赖性强**：实验显示需要数千条记录才能获得理想的提升效果（见论文图 6），这限制了其在小规模临床研究中的应用。
*   **真实场景挑战**：在人工模拟失真上表现优异，但在复杂的真实跨设备迁移中效果受限，说明真实世界的领域差异比单纯的噪声更复杂。
*   **计算开销**：虽然可以在笔记本上运行，但对于实时自适应来说，50 个 Epoch 的微调时间仍然过长。
