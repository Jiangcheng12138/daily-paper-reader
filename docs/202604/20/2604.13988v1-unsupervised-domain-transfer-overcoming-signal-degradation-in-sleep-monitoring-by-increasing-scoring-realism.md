## 摘要
目的：探讨睡眠图的“真实性”是否可用于指导一种无监督方法，以处理移动睡眠监测中任意类型的信号退化。方法：将预训练的最先进“u-sleep”模型与“判别器”网络相结合，我们将目标域的特征与预训练期间学习到的特征空间进行对齐。为了测试该方法，我们使用真实的信号退化对源域进行失真处理，以观察该方法对不同类型退化的适应能力。我们将所得模型的性能与针对每种迁移类型以有监督方式设计的最佳情况模型进行了比较。主要结果：根据失真类型的不同，我们发现无监督方法可以将 Cohen's kappa 系数提高 0.03 到 0.29，且对于所有迁移任务，该方法均未降低性能。然而，该方法从未完全达到估计的理论最优性能，且在针对两项睡眠研究之间的现实领域失配进行测试时，其收益并不显著。意义：“判别器引导的微调”是处理自然场景下睡眠监测信号退化的一种有趣且具有一定前景的方法。特别是，它对一般睡眠数据的揭示具有启发意义。然而，在将其投入实际应用之前，仍需进一步开发。

## Abstract
Objective: Investigate whether hypnogram 'realism' can be used to guide an unsupervised method for handling arbitrary types of signal degradation in mobile sleep monitoring.   Approach: Combining a pretrained, state-of-the-art 'u-sleep' model with a 'discriminator' network, we align features from a target domain with a feature space learned during pretraining. To test the approach, we distort the source domain with realistic signal degradations, to see how well the method can adapt to different types of degradation. We compare the performance of the resulting model with best-case models designed in a supervised manner for each type of transfer.   Main Results: Depending on the type of distortion, we find that the unsupervised approach can increase Cohen's kappa with as little as 0.03 and up to 0.29, and that for all transfers, the method does not decrease performance. However, the approach never quite reaches the estimated theoretical optimal performance, and when tested on a real-life domain mismatch between two sleep studies, the benefit was insignificant.   Significance: 'Discriminator-guided fine tuning' is an interesting approach to handling signal degradation for 'in the wild' sleep monitoring, with some promise. In particular, what it says about sleep data in general is interesting. However, more development will be necessary before using it 'in production'.

*   **研究动机**：传统的有监督微调需要大量人工标注的标签，这在实际应用中成本极高。作者提出：能否利用睡眠图（Hypnogram）本身的“真实性”（Realism）——即睡眠阶段演变的特定规律（如阶段持续时间、转换顺序等）——作为约束，在无需目标域标签的情况下，通过无监督方法修复受损信号的影响。

### 2. 方法论：核心思想与技术细节
*   **核心思想**：采用**对抗性领域迁移（Adversarial Domain Transfer）**框架。通过一个判别器来判断生成的睡眠评分序列是来自“干净的源域”还是“受损的目标域”。
*   **关键技术细节**：
    *   **模型结构**：基于 **U-Sleep** 架构。将其拆分为编码器（Encoder）和解码器（Decoder）。
    *   **判别器引导微调（Discriminator-guided fine tuning）**：
        *   **源域编码器 ($E_s$)**：冻结，提供干净信号的特征基准。
        *   **目标域编码器 ($E_t$)**：可学习，负责将受损信号映射到特征空间。
        *   **解码器 ($Dec$)**：冻结，确保输出空间的一致性。
        *   **判别器 ($D$)**：一个轻量级的 Transformer 架构，输入是睡眠阶段的概率分布序列，学习识别“真实”睡眠图的模式。
    *   **损失函数**：
        *   **对抗损失 ($L_{adv}$)**：目标编码器试图欺骗判别器，使其无法区分源域和目标域的特征。
        *   **锚点损失 ($L_{anchor}$)**：惩罚目标编码器与源编码器在相同输入下的偏差，以保持预训练的几何结构。
*   **算法流程**：通过最小-最大博弈，迫使模型在面对噪声信号时，依然能生成符合人类睡眠生理规律（即“真实感”）的评分结果。

*   **参数设置**：Batch Size 为 1（全记录处理），训练 50 个 Epoch。编码器学习率 $10^{-5}$，判别器 $10^{-6}$。

*   **实验规模**：使用了近 4000 晚的睡眠数据，规模较大。
*   **实验类型**：
    *   针对三种主要失真类型的参数化测试（如不同的滤波截止频率、不同的噪声通道）。
    *   **消融实验**：测试了锚点损失、判别器数量、Batch Size、位置编码等对性能的影响。

### 6. 主要结论与发现
*   **性能提升**：在所有人工模拟的失真场景下，该方法均显著优于预训练模型，Cohen's kappa 提升幅度在 **0.03 至 0.29** 之间。
*   **真实性有效**：实验证明，强制要求睡眠图符合“真实模式”确实能引导模型忽略噪声，从受损信号中提取有用信息。
*   **局限性**：虽然优于原始模型，但仍无法达到有监督训练的性能水平。在样本量较小的真实跨领域任务（Tabar 数据集）中，提升不显著。

### 7. 优点与亮点
*   **无需标签**：完全无监督，非常适合标注成本高昂的临床应用。
*   **通用性**：不针对特定噪声建模，而是通过判别器学习“什么是正确的输出”，从而应对多种类型的信号退化。
*   **生理启发**：模拟了人类专家在面对糟糕信号时，利用生理常识进行“合理推断”的过程。

### 8. 不足与局限
*   **数据依赖性**：从实验看，该方法需要数千条记录才能获得理想的提升效果，数据效率有待提高。
*   **真实迁移挑战**：在处理真实的、复杂的领域差异（而非人工模拟的噪声）时，表现尚不稳定。
*   **架构限制**：目前仅基于 U-Sleep 验证，对于其他如基于频谱输入的模型（如 SleepTransformer）的效果尚不明确。
*   **性能上限**：无法完全替代有监督学习，在极度恶劣的信号下改善有限。
