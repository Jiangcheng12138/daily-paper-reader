## 摘要
目的：探讨睡眠图的“真实性”是否可用于指导一种无监督方法，以处理移动睡眠监测中任意类型的信号退化。方法：将预训练的最先进“u-sleep”模型与“判别器”网络相结合，我们将目标域的特征与预训练期间学习到的特征空间进行对齐。为了测试该方法，我们使用真实的信号退化对源域进行失真处理，以观察该方法对不同类型退化的适应能力。我们将所得模型的性能与针对每种迁移任务以有监督方式设计的最佳情况模型进行了比较。主要结果：根据失真类型的不同，我们发现无监督方法可以将 Cohen's kappa 系数提高 0.03 到 0.29 不等，且对于所有迁移任务，该方法均未降低性能。然而，该方法从未完全达到估计的理论最优性能，且在针对两项睡眠研究之间的现实领域失配进行测试时，其收益并不显著。意义：“判别器引导的微调”是处理“自然场景下”睡眠监测中信号退化的一种有趣且具有一定前景的方法。特别是，它对一般睡眠数据的揭示具有重要意义。然而，在将其投入“实际应用”之前，仍需进一步开发。

## Abstract
Objective: Investigate whether hypnogram 'realism' can be used to guide an unsupervised method for handling arbitrary types of signal degradation in mobile sleep monitoring.   Approach: Combining a pretrained, state-of-the-art 'u-sleep' model with a 'discriminator' network, we align features from a target domain with a feature space learned during pretraining. To test the approach, we distort the source domain with realistic signal degradations, to see how well the method can adapt to different types of degradation. We compare the performance of the resulting model with best-case models designed in a supervised manner for each type of transfer.   Main Results: Depending on the type of distortion, we find that the unsupervised approach can increase Cohen's kappa with as little as 0.03 and up to 0.29, and that for all transfers, the method does not decrease performance. However, the approach never quite reaches the estimated theoretical optimal performance, and when tested on a real-life domain mismatch between two sleep studies, the benefit was insignificant.   Significance: 'Discriminator-guided fine tuning' is an interesting approach to handling signal degradation for 'in the wild' sleep monitoring, with some promise. In particular, what it says about sleep data in general is interesting. However, more development will be necessary before using it 'in production'.

### 2. 论文提出的方法论
*   **核心思想**：采用**对抗性领域迁移（Adversarial Domain Transfer）**框架。将睡眠评分任务视为特征对齐问题，通过判别器强制让受损信号生成的特征分布与干净信号生成的特征分布保持一致。
*   **关键技术细节**：
    *   **模型结构**：基于 **U-Sleep** 架构。将其拆分为编码器（Encoder）和解码器（Decoder）。
    *   **双编码器设计**：设置一个冻结的源域编码器（$E_s$）处理干净数据，一个可学习的目标域编码器（$E_t$）处理失真数据。
    *   **判别器（Discriminator）**：一个轻量级的 Transformer 二进制分类器，输入是模型预测的睡眠阶段概率分布。它的任务是区分该睡眠图是来自“干净信号”还是“失真信号”。
    *   **损失函数**：
        1.  **对抗损失（Adversarial Loss）**：目标域编码器尝试欺骗判别器，使其认为失真信号生成的睡眠图是真实的。
        2.  **锚点损失（Anchor Loss）**：惩罚 $E_t$ 与 $E_s$ 在相同输入下的特征偏差，确保模型在微调过程中不会丢失预训练的几何特征。
*   **算法流程**：在无标注的目标域数据上，通过最小化对抗损失和锚点损失的加权和，仅更新目标域编码器的参数。

### 6. 主要结论与发现
*   **性能提升**：在所有人工失真场景下，无监督微调均显著提升了性能。Cohen's kappa 的提升幅度在 **0.03 到 0.29** 之间。
*   **真实性引导有效**：实验证明，通过强制要求睡眠图符合“真实规律”，确实可以引导模型忽略噪声或绕过信号退化。
*   **未达上限**：虽然优于原始模型，但该无监督方法仍无法达到有监督训练的理论最优性能。
*   **数据依赖性**：性能提升与训练数据量呈正相关，通常需要数千条记录才能获得理想的改进。

### 7. 优点
*   **完全无监督**：不需要目标域的任何人工标注，这对于实际部署非常友好。
*   **通用性强**：不针对特定噪声建模（如不尝试去噪），而是通过特征对齐处理“任意”类型的退化。
*   **科学洞察**：利用了睡眠图空间的“稀疏性”和“规律性”，这为生物信号处理提供了一个新颖的视角。

### 8. 不足与局限
*   **真实场景失效**：在处理真实的跨数据集迁移时效果不佳，可能是因为真实领域的差异比人工模拟的失真更复杂，或者受限于目标域数据量（仅 120 条记录）。
*   **数据效率低**：目前需要大量的无标注数据（数千条记录）才能有效微调，限制了其在小规模临床研究中的应用。
*   **计算成本**：虽然在笔记本上可运行，但对于实时性要求高的场景，50 个 Epoch 的微调可能过慢。
*   **偏差风险**：如果目标人群的睡眠模式本身就极不规律（如严重睡眠障碍患者），判别器可能会错误地将真实的异常模式视为“不真实”而予以纠正。
