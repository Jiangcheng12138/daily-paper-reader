## 摘要
通过多导睡眠图诊断睡眠呼吸暂停仍然资源密集，且依赖于耗时的手动数据分析和评分。最近的研究表明，睡眠呼吸暂停事件的中枢神经系统效应可通过脑电图（EEG）信号检测。然而，大多数工作使用单一特征类型对不同的数据集结合不同的分类算法。在本工作中，我们针对一个儿科受试者数据集，从多通道EEG中自动检测睡眠呼吸暂停，对深度学习架构和特征表示进行了全面比较。我们评估了视觉变换器和图注意力网络在不同信号表示上的表现：原始时域信号、短时傅里叶变换频谱图、基于相干性的图，以及两种拓扑数据分析（TDA）衍生特征。通过训练集和测试集的年龄与性别匹配，我们在2410名儿科受试者上训练，并在575名儿科受试者上测试。使用基于TDA特征训练的视觉变换器模型，我们达到了0.750的最佳测试AUC。按患者人口统计学（年龄、性别、AHI严重程度）和睡眠阶段（N1、N2、N3、REM）进行分层分析，揭示了显著的性能差异。我们的结果证明了基于EEG的自动OSA筛查的可行性，同时突显了临床部署中的关键挑战。

## Abstract
Sleep apnea diagnosis via polysomnography remains resource intensive and relies on time consuming manual data analysis and scoring. Recent work has demonstrated that central nervous system effects of sleep apnea events can be detected through electroencephalogram (EEG) signals. However, most work uses a single feature type on various datasets combined with different classification algorithms. In this work, we present a comprehensive comparison of deep learning architectures and feature representations for automated sleep apnea detection from multichannel EEG on a single dataset of pediatric subjects. We evaluate Vision Transformers and Graph Attention Networks across distinct signal representations: raw temporal signals, short-time Fourier transform spectrograms, coherence based graphs, and two topological data analysis (TDA) derived features. Using age and sex matching of our train and test sets, we train on 2410 pediatric subjects and test on 575 pediatric subjects. We achieve a best test AUC of 0.750 using a vision transformer based model trained on TDA features. Stratified analysis across patient demographics (age, sex, AHI severity) and sleep stages (N1, N2, N3, REM) reveals significant performance variation. Our results demonstrate the feasibility of EEG based automated OSA screening while highlighting essential challenges for clinical deployment.

## 2. 论文提出的方法论

### 核心思想
通过特征提取+深度学习模型，从30秒7通道EEG信号中判断是否存在睡眠呼吸暂停（定义为该窗口内异常呼吸事件覆盖≥75%）。对比不同特征表示对模型性能的影响，并分析模型在不同亚组中的表现差异。

### 关键技术细节

#### 特征提取方法
1. **原始EEG信号**：对7通道、256Hz、30秒窗信号进行60Hz/120Hz陷波滤波，得到形状 `7×7681` 的数组。
2. **短时傅里叶变换（STFT）**：对每个通道计算STFT（Hann窗，129个频率bin，62个时间bin），取对数幅度后Z归一化，得到形状 `7×129×62`。
3. **相干性图**：基于信号相干性构建7节点全连接图。节点特征为各通道STFT，边权重为两通道间的幅度平方相干性矩阵（形状 `7×7×129`）。
4. **拓扑数据分析（TDA）特征**：
   - 由相干性矩阵得到差异性矩阵 `D = 1 - C`。
   - 使用Rips复形对差异性矩阵进行持久同调，提取H0和H1同调群的持久性图。
   - 定义**寿命熵持久曲线**，然后使用**Hermite展开（HEPC）**和**任意周期傅里叶近似（AP-FAPC）**将持久曲线向量化（各16个系数）。最终HEPC+AP-FAPC串联后每个同调群/频率bin获得48个系数，形状 `2×129×48`。
   - HEPC仅使用HERMITE系数，形状 `2×129×16`。

#### 模型架构
1. **Vision Transformer（ViT）**：输入先实例归一化，然后提取patches（2D输入16×16，1D输入256）。每个patch线性投影至1024维，加入可学习位置编码。主干为24个Transformer块，多头自注意力（16头），最终全局平均池化后接MLP分类头。适用于除相干图外的所有特征。
2. **Graph Attention Network（GAT）**：用于相干图特征。边特征经全连接编码，节点特征投影至512维。堆叠4个GATv2层（每层8个注意力头，隐藏维512），使用注意力池化聚合节点表示后接分类器。

#### 训练策略
- 优化器：Adam；损失函数：逆频率类权重（未调整正负样本比例但加权损失）。
- 超参数搜索：网格搜索。
- 学习率：ViT+时序/STFT为1e-5，ViT+HEPC为1e-4，GAT为1e-6。
- 批次大小：ViT用256，GAT用1024。
- 训练轮数：10 epochs，使用one-cycle学习率调度。
- 数据划分：按受试者分层（年龄、性别）进行75%/25%训练/测试分裂，保证各群体比例一致。

## 6. 论文的主要结论与发现

1. **最佳性能**：使用Vision Transformer+HEPC特征，测试AUC达0.750，平衡准确率0.685。紧接其次为GAT+相干图（AUC 0.748）。原始EEG基线最差（AUC 0.639）。
2. **特征有效性**：TDA特征（HEPC）优于传统时频表示（STFT），证明持久同调能较好捕捉睡眠呼吸暂停相关的功能连接变化。
3. **参数效率**：GAT模型参数数仅为ViT的~3%（9.98M vs 308M），但性能接近，说明显式编码通道关系比自注意力更高效。
4. **HEPC+AP-FAPC退化**：加入AP-FAPC系数后性能略降（AUC 0.740），提示额外拓扑复杂性引入噪声。
5. **性别差异**：所有模型在女性受试者上表现优于男性，且该差异不能由年龄或AHI分布差异解释（统计检验无显著差异），暗示EEG信号或OSA表现存在性别固有差异。
6. **年龄波动**：模型在8岁、10岁达到峰值，2-6岁和13-18岁性能较差，提示年龄特异性EEG模式。
7. **严重程度影响**：性能在健康和轻度OSA上最高，中度和重度下降（部分模型重度回升），但HEPC重度和中度相近。表明模型对严重病例识别不佳。
8. **睡眠阶段依赖性**：
   - N2和N3睡眠AUC最高，但呈现出“高灵敏度低特异度（轻睡眠Wake/N1）和低灵敏度高特异度（深睡眠N3）”的相反模式。
   - 表明模型主要检测与**觉醒相关的呼吸暂停**（arousal-associated apnea）而非纯呼吸道阻塞，因为觉醒阈值的生理差异直接反映在EEG信号上。

## 8. 不足与局限

- **单一数据集与单次分裂**：仅使用NCHSDB儿科数据集，未在成人数据或外部儿科中心数据上验证，模型泛化能力未知；仅用一个训练/测试分裂，结果可能存在随机波动，缺乏K折交叉验证。
- **性能不足临床部署**：最佳AUC仅0.750，远低于临床实用阈值（通常要求>0.9），且模型在中等/重度OSA、特定年龄和性别上表现更差，距离实际应用有较大差距。
- **未消融模型组件**：未研究不同patch大小、层数、注意力头数等对性能的影响，也缺少传统机器学习基线（如SVM、随机森林）对比，无法评估深度学习相对于简单方法的增益程度。
- **类别不平衡问题**：正样本率仅3.36%，虽然使用了逆频率权重，但模型仍可能偏向多数类（特异度普遍较低），且未使用F1-score或PR曲线等对不平衡更鲁棒的指标。
- **忽略时间依赖性**：模型独立处理30秒窗口，未利用连续时间上下文（如循环神经网络或序列模型），可能遗漏睡眠结构演变信息。
- **未报告计算资源**：缺少GPU型号、训练时间等信息，影响可重复性和算力成本评估。
- **未进行事件级检测**：模型做30秒窗口的分类，而不是逐呼吸事件检测，与临床事件计数（AHI）存在差距。
- **HEPC+AP-FAPC组合未提升**：作者未深入分析原因，仅归因于噪声，缺乏进一步实验（如单独AP-FAPC、降维等）。
- **正负标签定义局限**：使用“窗口覆盖75%事件”为阳性，可能引入边界模糊（如事件部分在窗口内），且未区分事件类型（呼吸暂停/低通气/氧减）对EEG影响可能不同。
- **偏差风险**：模型在女性上表现更好，可能隐含对男性患者漏诊的风险；对特定年龄的低性能可能导致系统性的不平等影响。
