## 摘要
自动睡眠分期是被动脑机接口（pBCI）的基本应用，它解码自发神经状态以实现独立于用户意图的闭环干预。本研究评估了基于去趋势波动分析（DFA）的临界性特征，用于深度睡眠（N3）的特异性识别。我们分析了来自290名老年女性的347,232个脑电周期，使用UMAP流形学习可视化状态转换。随后，通过10折交叉验证对六个分类器进行基准测试，使用平衡准确率确定用于神经反馈的最佳“状态感知”引擎。朴素贝叶斯达到了最高的平均平衡准确率（87.17%±0.24%），显著优于全连接深度神经网络（FNN：81.58%）和随机森林（80.97%）。线性模型（LDA：57.21%；SVM：51.01%）表现较差，表明DFA衍生的临界性特征位于独特的非线性流形上。对脑电临界性的概率解码为pBCI提供了高精度的感知机制。这种稳健的分类流程支持开发状态依赖的神经反馈，例如靶向听觉刺激，以增强认知恢复。

## Abstract
Automated sleep staging is a fundamental application of passive Brain-Computer Interfaces (pBCI), decoding spontaneous neural states to enable closed-loop interventions independent of user intent. This study evaluates criticality features derived from Detrended Fluctuation Analysis (DFA) for the specific identification of deep sleep (N3).   We analyzed $347,232$ EEG epochs from $290$ older women using UMAP manifold learning to visualize state transitions. Subsequently, six classifiers were benchmarked via 10-fold cross-validation, using balanced accuracy to determine the optimal "state-sensing" engine for neurofeedback.Naive Bayes achieved the highest mean balanced accuracy ($87.17\% \pm 0.24\%$), significantly outperforming a fully connected deep neural network (FNN: $81.58\%$) and Random Forest ($80.97\%$). Linear models (LDA: $57.21\%$; SVM: $51.01\%$) performed poorly, indicating that DFA-derived criticality features reside on a distinct, non-linear manifold.   Probabilistic decoding of EEG criticality provides a high-accuracy sensing mechanism for pBCIs. This robust classification pipeline supports the development of state-dependent neurofeedback, such as targeted auditory stimulation, to enhance cognitive recovery.

- **核心问题**：如何实现高精度、自动化的深度睡眠（N3期）分类，以服务于被动脑机接口（pBCI）框架下的闭环睡眠改善神经反馈（如听觉刺激）。
- **背景**：自动化睡眠分期被视为pBCI的“教科书级”应用——它利用自发的脑电活动（无用户意识意图），输出状态估计而非直接命令。传统的睡眠分期依赖睡眠纺锤波、K复合波等标记，而本研究提出利用脑电信号临界性（criticality）——即神经系统处于有序与无序相变点附近的特性——作为特征，通过去趋势波动分析（DFA）提取长程时间相关性（LRTC）的标度指数，用以区分深度睡眠与其他睡眠阶段。
- **含义**：为居家睡眠干预、认知健康增强提供高精度的“状态感知”引擎，支持自主、无感的闭环神经反馈系统。

## 2. 论文提出的方法论

### 核心思想
- 利用脑电信号临界性（通过DFA计算Hurst指数 \( H \)，即标度指数 \( \alpha \)）作为单一特征，捕捉N3期深睡眠特有的长程有序动力学，与其它睡眠阶段（清醒、N1、N2、REM）的随机或碎片化动态相区分。
- 假设：健康大脑在深度睡眠期间更接近临界点，而疾病状态（如认知衰退）会破坏这一特征，但N3期临界性增强的判别力仍存在。

### 关键技术细节
- **DFA计算（四步流程）**：
  1. **积分**：将原始时间序列 \( x(i) \) 转换为累积剖面 \( Y(i) = \sum_{k=1}^{i} [x(k) - \langle x \rangle] \)。
  2. **分段**：将剖面分割为 \( N_n = \text{int}(N/n) \) 个非重叠段，每段长度 \( n \)。
  3. **局部去趋势**：对每段用一阶多项式（\( k=1 \)）拟合趋势并减去，消除非平稳漂移。
  4. **波动函数计算**：取 \( q=2 \)（标准DFA），计算均方根波动 \( F(n) = \sqrt{ \frac{1}{2N_n} \sum_{\nu=1}^{2N_n} [F^2(\nu, n)] } \)。
- **标度指数**：由 \( \log F(n) \) 对 \( \log n \) 的斜率得到，\( \alpha \approx 0.5 \) 为白噪声，\( \alpha \approx 1.0 \) 为 \( 1/f \) 噪声（临界性特征）。
- **特征处理**：对4个EEG通道（C3A2、C4A1、C3A1、C4A2）分别计算每个30秒epoch的Hurst指数，构成4维特征向量。
- **流形可视化**：无监督UMAP（统一流形逼近与投影）将347,232个epoch的4维特征嵌入3维空间，观察N3期的聚类结构。
- **分类器对比**：6种分类器，涵盖线性、概率、集成、神经网络范式：
  - 线性模型：LDA、线性SVM。
  - 非线性/集成：随机森林（100棵树，最大深度10）、K近邻（k=5）。
  - 概率/神经网络：高斯朴素贝叶斯、四层全连接神经网络（FNN，3个100神经元隐藏层+1个50神经元层，ReLU激活）。

## 3. 实验设计

### 数据集
- **来源**：国家睡眠研究资源（NSRR）中的“骨质疏松性骨折研究（SOF）”子集。
- **被试**：290名美国老年女性（≥65岁），MMSE>24，5年后有随访3MS评估。
- **认知分组**：正常（3MS≥88）、轻度认知障碍MCI（78-88）、痴呆（<78）。
- **睡眠数据**：家庭便携式多导睡眠监测（Compumedics P-Series），4通道EEG（C3、C4、A1、A2），128 Hz采样，30秒epoch手动分型（R&K标准，N3/N4合并为N3）。
- **样本量**：共347,232个epoch，覆盖所有睡眠阶段及清醒状态（pre-sleep、post-sleep、夜间清醒）。

### Benchmark与对比方法
- **任务**：二分类N3 vs. 非N3（所有其他阶段合并）。
- **评价指标**：平衡准确率（Balanced Accuracy），采用10折分层交叉验证（Stratified 10-fold CV），每折内使用StandardScaler标准化特征。
- **对比方法**：6个分类器（如上所述），包括深度神经网络作为强对标。

### 实验结果摘要
- **朴素贝叶斯**：87.17% ± 0.24%（最高）
- **FNN**：81.58% ± 0.68%
- **随机森林**：80.97% ± 0.41%
- **KNN**：80.35% ± 0.39%
- **LDA**：57.21% ± 0.16%
- **线性SVM**：51.01% ± 0.14%
- 此外，UMAP嵌入显示N3期在3D空间中形成紧凑且与其他阶段线性不可分的簇；Hurst指数在N3期显著高于所有其他阶段（Bonferroni校正p≤0.0001），且跨认知组一致。

## 6. 主要结论与发现

- **核心发现**：基于EEG临界性的DFA特征（Hurst指数）能够可靠地区分深度睡眠（N3）与其他睡眠阶段，且这一判别力在认知正常、MCI、痴呆三组中均保持一致。
- **最优分类器**：朴素贝叶斯取得最高平衡准确率87.17%，显著优于深度神经网络（81.58%）和随机森林（80.97%），暗示特征维度低且条件独立的假设适用。
- **线性模型失效**：LDA（57.21%）和线性SVM（51.01%）接近随机，验证了UMAP发现的N3特征流形非线性不可分的性质。
