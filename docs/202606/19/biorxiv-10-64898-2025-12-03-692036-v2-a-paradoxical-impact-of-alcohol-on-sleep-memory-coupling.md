## 摘要
睡眠在记忆巩固中起着基础性作用，但这可能受到生理状态的影响。急性乙醇摄入深刻影响后者，但醉酒是否影响睡眠在记忆巩固中的作用仍未被探索。在这里，我们证明急性乙醇翻转了果蝇中睡眠与记忆之间的典型联系。通常，饱食的果蝇需要睡眠来进行食欲性记忆巩固，但必须觅食的饥饿果蝇则转向不依赖睡眠的记忆。乙醇通过引起向不依赖睡眠的记忆的转换，选择性地损害饱食果蝇的记忆巩固，随后由于乙醇诱导的镇静作用，这种记忆无法得到支持。在这些条件下，睡眠剥夺能够挽救记忆，并且需要神经肽F的上调，该肽通过PPL1多巴胺能神经元上的NPF受体传递信号。我们认为由饥饿或乙醇诱导的奖赏寻求行为会引发不依赖睡眠的记忆。然而，乙醇具有矛盾的影响，它诱导记忆途径的转换，但同时通过偏好途径抑制巩固，从而使睡眠对记忆巩固变得有害。

## Abstract
Sleep serves a fundamental role in memory consolidation, but this can be influenced by physiological context. The latter is impacted profoundly by acute ethanol consumption, but whether intoxication affects the role of sleep in memory consolidation remains unexplored. Here, we demonstrate that acute ethanol inverts the canonical link between sleep and memory in Drosophila. Typically, satiated flies require sleep for appetitive memory consolidation, but starved flies that must forage for food switch to sleep-independent memory. Ethanol selectively impairs memory consolidation in satiated flies by causing a switch to sleep-independent memory, which then cannot be supported because of ethanol-induced sedation. Under these conditions, sleep deprivation rescues memory and requires the upregulation of neuropeptide F, which signals through NPF receptors on PPL1 dopaminergic neurons. We suggest that reward-seeking behaviors, induced by starvation or ethanol, invoke sleep-independent memory. However, ethanol has a paradoxical impact, wherein it induces a switch in memory pathways but concurrently suppresses consolidation through the preferred pathway, thereby causing sleep to become detrimental for memory consolidation.
